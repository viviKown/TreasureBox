## CSS 定义

由两个主要的部分构成：选择器（实际上就是HTML标签，元素中设置"id" 和 "class"选择器。），以及一条或多条声明:

![Alt text](https://github.com/viviKown/TreasureBox/blob/master/img/css_struct.png)

样例：

（1）h1  标签类

```css
h1{
    text-align:center;
	color:red;
}
```

 （2）#para1    id引用类

```css
#para1{
text-align:center;
color:red;
} 
```

（3）.center   class引用类

```css
.center 
{
text-align:center;
}
```

引用例子：

```
<h1>标题居中</h1>
<h2 id="center">标题居中</h2>
<p class="center">段落居中。</p> 
```
